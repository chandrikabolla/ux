Cavas Health Record System - User Experience

CAVAS is a global patient health record system that allows hospitals to digitally store patient medical data and patients to book appointments, access their diagnosis, medical information using their password protected accounts.

People

Chandrika Bolla – a team of 5

Process

  • Followed top down approach from “The Elements of User Experience” – by Jessee James Garrett
  • Conducted User Research Interviews to know already existing problems by the patients
  • Practiced Card Sorting using almost 10 users to build Information architecture of the website, remote card sorting using optimalworkshop.com
  • Designed storyboards of the scenarios to better understand the user scenarios
  • Created personas after analyzing the users from the interviews
  • Designed Wireframing (Low Fidelity prototype) as the step of designing elements on the screen
  • Prototyped (High Fidelity) the whole site using Axure RP
  • Conducted A/B Testing to analyze user’s views about variations in design and evaluate the impact of the changes done
  • Developed usability test scenarios and tasks (poor and better tasks) to accomplish user goals
  • Conducted Usability Testing using 20 users using Usability test scripts

Methodologies implemented for UX Research

User Research interview template, wireframes, prototype screens, usability task scenarios, user tasks, usability test script

Technology

Axure RP, SurveyMonkey, Moqups.com
